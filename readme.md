## Installation

install aplikasi ini menggunakan bantuan dari NPM

```bash
  git clone https://github.com/Cloud-Dark/Witch_Saga
  cd Witch_Saga
```
Jalankan aplikasi
```bash
  php /withoop/form.php
```
